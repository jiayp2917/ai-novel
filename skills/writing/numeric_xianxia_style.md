# 《机制怪？我是数值怪》全民修仙正文风格

## 风格总目标

正文面向轻松搞笑爽文读者。核心阅读感是：现代全民修仙时代，别人认真研究灵根、术法、阵法、秘境和考核规则，许满凭离谱肉身数值和朴素理解，把严密机制撞成大型笑话。写作时优先保证清楚、顺滑、有推进感；爽点来自行为结果、规则误伤、外界脑补和喜剧反差，不靠作者旁白解释“这里很爽”。

## 个人节奏迁移

沿用旧素材中“短段落、高对白、强场景推进”的写法。章节正文以一两句一段为主，中位段落控制在 20 字左右，连续长段只用于关键说明或情绪压迫，不能成为常态。每章要有清晰推进链：进入场景 -> 规则或目标出现 -> 人物快速反应 -> 行动导致变化 -> 留一个可继续追读的结果。

旧素材只迁移节奏、断句、对白推动和围观反应；不迁移旧题材里的灾变压迫、面板弹窗、道具数值表、怪物危机、背叛复仇和清单式播报。规则判定用试炼石碑、阵盘、监考长老、天机榜、留影复核承接，不写成游戏式播报。

## 本书样段校准

本书笑点适合用“先抬高可靠性，再被许满朴素执行打穿”的结构。先写测试人偶、阵法、秘境、榜单、试炼规则有多稳，再让许满用一句憨直问题破掉预期，例如担心打坏、没听清目标、认真问长老怎么办。旁人先不信，再用口语吐槽承接反差；最后用场地、人偶、榜单或长老反应给出实锤。

许满的口吻要憨直、实在、反应慢半拍。可少量使用“来了来了”“打坏怎么办”“俺当时没听到”这类口语，但不要写成故意卖傻。他不是装疯卖傻，也不是故意挑战规则；他只是认真按自己理解做。

轻喜剧余波不要靠作者总结，要靠旁人复述荒诞点、负责人黑脸、熟人拆台和外界误读来发酵。围观者可以用一句话点破笑点，例如“谁家好人闭着眼闯幻阵，还念我看不见”。负责阵法或测试的人要先确认设备没坏，再被现实打脸；熟人可以用外号、旧梗、半损半熟的调侃收尾。

## 对白规则

对白占比可以偏高，但每句都要推动信息、误解或喜剧反差。王大雷负责嘴快和吐槽，吐槽要具体，可以先自嘲、再吐槽许满，重点是说出读者心里那句“这也行”。林浅偏理性，她会先相信阵法和数据，随后发现数据“不属于它应该承受的一切”。李燃统一使用这个名字；他是传统天才对照，嘴硬、爱面子，看到许满登顶后先不服，再偷偷研究原因。

## 许满写法

许满性格憨厚、实诚、迟钝，不主动装逼，不刻意钻空子，不嘲讽别人。他面对规则时先按字面和常识理解：不能飞就跑，守住旗就别让旗丢，不能动阵眼就把旁边压稳，目标是去对岸就拖着船走。他的强来自肉身数值异常，包括力量、速度、耐力、抗性和恢复，而不是临时开挂或突然聪明。写许满时要保留认真劲：他是真的在训练、通关、听话，只是结果过于离谱。

## 配角写法

林浅是天才阵修少女，也是规则解释者和轻微暧昧线。她的变化是从“我能算明白”到“这个人本身就是变量”。写暧昧要轻，不抢主线爽点。

王大雷是室友和传播者，负责把名场面变成留影热梗。他可以惊讶、吐槽、替读者发问，但要有具体观察，不要沦为重复弹幕。

李燃是传统天才对照。他擅长术法和战术，前期不服许满，后期嘴硬但偷偷研究“许式通关”。他的作用是衬托传统解题法与数值破局的反差，不写成沉重反派。

王腾、石三这类新名字只有在大纲或当前章节已提供时才能使用。若没有明确设定，不要自动新增他们；可以把“阵法负责人黑脸”和“熟人拆台”的关系转移给已有角色。

## 爽点规则

每个爽点尽量形成“规则建立 -> 众人认真解题 -> 许满朴素执行 -> 机制承受不了 -> 外界反应发酵”的链条。秘境、阵法、考核、赛事规章不是敌人，只是默认修士数值在正常范围内；许满让这些默认条件失效。后期爽点要升级表现形式：场地误判、阵盘复核失败、长老临时改安全指引、天机榜措辞越来越谨慎、观众从震惊变成脑补。

## 禁止事项

不要新增没有上下文来源的势力、境界、技能、道具和人物关系。不要写苦大仇深、阴谋黑手、背叛牺牲等沉重线。不要让许满突然变成精明算计型主角。不要让任何统一意志主动针对许满。不要把正文写成大纲、设定说明、章节总结或作者吐槽。

## 生成前自检

输出前检查：本章是否有明确场景目标；许满是否仍然憨厚实诚；规则来源是否来自秘境、阵法、考核或赛事本身；爽点是否由具体行为和后果产生；配角反应是否具体；是否只输出可直接使用的单章正文。
